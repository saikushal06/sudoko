📖 About the Game

Sudoku is a classic logic-based puzzle game played on a 9×9 grid.
The objective is to fill the grid so that:

Each row contains numbers 1-9

Each column contains numbers 1-9

Each 3×3 subgrid contains numbers 1-9

No number should repeat within a row, column, or 3×3 box.

The game challenges players to use logical reasoning and pattern recognition to complete the puzzle.

🎮 How to Play

Select any empty cell in the grid

Choose a number from 1–9

Ensure the number does not repeat in:

the same row

the same column

the same 3×3 box

Continue filling the grid until the puzzle is complete.
